# Pythtris

Tetris made with Tkinter in Python

<img src="assets/images/gameplay.gif" width="250">

## How to play
| Command      | Key                  |
|--------------|----------------------|
| Move left    | Left arrow           |
| Move right   | Right arrow          |
| Snap left    | a or A               |
| Snap right   | d or D               |
| Rotate right | Up arrow             |
| Soft drop    | Down arrow           |
| Hard drop    | s or S               |
| New Game     | Ctrl + n or Ctrl + N |
| Pause/Resume | Esc                  |

## Score board
| Action       | Score        |
|--------------|--------------|
| Single       | 40 * level   |
| Double       | 100 * level  |
| Triple       | 300 * level  |
| Tetris       | 1200 * level |
| Entire Board | 2000         |
